**关于我们**

我们正在构建下一代预测市场平台，让用户能够对真实世界事件进行交易与定价。产品融合了金融交易系统的高性能要求与区块链的透明结算能力，团队节奏快、注重用户体验。现诚邀一位具备扎实前端功底、对交易型产品有热情的工程师加入我们。

**职位概述**

你将负责设计、开发并维护预测市场产品的用户界面，核心挑战包括：**高频数据驱动的交易界面、订单簿与 AMM 交互、实时行情展示、复杂状态管理与性能优化**。你将与产品、设计、后端工程师紧密协作，把复杂的交易逻辑转化为简洁直观的用户体验。

**主要职责**

- 开发、测试并维护预测市场产品的前端代码，构建交易、市场浏览、持仓管理等核心页面。
- 使用 React.js / Next.js 构建高性能、可复用的组件体系。
- 使用 TailwindCSS 与 MUI (Material UI) 搭建统一、美观的 UI 设计系统。
- 设计并落地复杂状态管理与缓存策略（React Query、Zustand 等），保证数据密集型页面的响应速度与一致性。
- 构建订单簿、行情图表、市场卡片等交易类 UI 组件，处理实时数据流与高频更新。
- 与后端工程师协作，对接 REST API 与 WebSocket，实现市场数据、订单状态的实时同步。
- 持续优化页面性能、加载速度与交互流畅度。
- 保证跨浏览器兼容与良好的移动端体验。
- 参与从需求、开发、上线到迭代的完整产品生命周期；参与代码评审，指导初级工程师。

**任职要求**

- 5 年以上前端开发经验。
- **精通 JavaScript 与 TypeScript**。
- **深度使用 React.js 与 Next.js**，具备组件化架构设计能力。**熟练使用 TailwindCSS** 进行快速样式开发，熟悉工具链配置与自定义主题。
- **熟练使用 MUI (Material UI)**，能基于 MUI 进行深度定制与主题扩展。
- **深入理解现代状态管理模式与缓存策略**，熟练使用 React Hooks/Context、React Query、Zustand、Mobx 或 Recoil 等方案。
