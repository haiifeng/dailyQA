flex: 1 0 auto 分别代表什么
flex:0 = 0 1 0% 表示元素的尺寸自适应且不会被拉伸或收缩；
flex:none = 0 0 auto表示元素的尺寸固定且不会被拉伸或收缩；
flex:1 = 1 1 0% 表示元素的尺寸根据父容器的剩余空间分配，并且会被拉伸或收缩以填充剩余空间；
flex:auto 1 1 auto 表示元素的尺寸自适应，并且可以被拉伸或收缩以填充可用空间。
